# AI-Volumen-Control
Project with artificial intelligence to control the volume with the opencv module

## Prerequisites
modules cv2, mediapipe, numpy, ctypes, comtypes, pycaw.

## Run

To run execute VolumeHandControl.py file

```sh
python3 VolumeHandControl.py
```
